# Harmony
## Information
* Shortened url service with full CICD
* https://vivien.fun/

## Techs & requirements
* HTML, CSS, JS
* Python, Flask
* Redis
* Nginx
* CICD
    * Docker/Docker compose
    * Github Action
    * GCP
        * Artifact Registry
        * Compute Engine

## Try this yourself
* docker compose up --detach
* Open url 127.0.0.1
* Insert any url, and get shortened url.
